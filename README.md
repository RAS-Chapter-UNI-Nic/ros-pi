# ros-pi

# Install RPi.GPIO Python Library

![image](https://docs.microsoft.com/en-us/windows/iot-core/media/pinmappingsrpi/rp2_pinout.png)


```
sudo apt-get update
sudo apt-get install rpi.gpio

```
